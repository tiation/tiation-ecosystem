API Documentation

Our APIs are designed with RESTful principles and include comprehensive authentication, rate limiting, and detailed error handling.

Authentication API

JWT-based authentication with refresh tokens and role-based access control.

curl -X POST https://api.tiation.com/auth/login \
  -H "Content-Type: application/json" \
  -d '{"email":"user@example.com","password":"secure_password"}'
View Authentication Docs →

Dice Rolling API

Comprehensive dice rolling with support for all D&D dice types and custom modifiers.

curl -X POST https://api.tiation.com/dice/roll \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"dice":{"d20":1,"d6":2},"modifier":5}'
View Dice API Docs →

Subscription API

Stripe-integrated subscription management with webhook support.

curl -X GET https://api.tiation.com/subscriptions/current \
  -H "Authorization: Bearer YOUR_TOKEN"
View Subscription Docs →
